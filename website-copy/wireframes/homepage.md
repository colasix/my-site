## VISUAL DESIGN GUIDANCE

**Hero Section:**
- Minimal, professional aesthetic. No overwhelming photography.
- White/light background with clear typography hierarchy.
- CTA button high contrast and immediately obvious.
- Optional: Subtle abstract visual or split-panel concept (not photos).

**Rapid Qualification Cards:**
- Three columns (desktop); stacked (mobile).
- Equal visual weight; equal-height cards.
- Clear icons or colour coding to distinguish scenarios.
- Each card leads to a different destination (no form submission).

**Services Overview Cards:**
- Three columns (desktop); stacked (mobile).
- High visual priority; take up significant vertical space.
- Clear headings, 2–3 bullet points per card.
- CTA button clearly visible on each card.

**Principals Section:**
- Side-by-side layout (desktop); stacked (mobile).
- Professional headshots (150–200px).
- Short bio text beside each photo.
- Clean typography; avoid overly styled design.

**Blog Feature:**
- Card layout with optional featured image per post.
- Title, excerpt, metadata clearly visible.
- Consistent sizing; 2–3 columns (desktop).

**Closing CTA:**
- Large, high-contrast button.
- Supporting text above button for context.
- Same styling as header CTA (consistency).

**Footer:**
- Four-column layout (Company | Services | Discovery & Diagnostic | [Brand/Contact Info]).
- CTA button prominently placed before company info.
- Links in secondary visual weight.
